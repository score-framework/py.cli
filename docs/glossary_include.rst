.. _cli_glossary:

.. glossary::

    shell command
        A click_ command that was registered for use with :mod:`score.cli`. The
        ``conf`` part in the following invocation is the :term:`shell command`:

        .. code-block:: console

            $ score conf list

        See :ref:`cli_command_aggregation` for details.

        .. _click: http://click.pocoo.org
